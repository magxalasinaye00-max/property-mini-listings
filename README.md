# property-mini-listings

A lightweight Vue 3 single-page prototype for Homes & Beyond — short-term property rentals in Cape Town.

## Features
- Dynamic property cards from local data.
- Search by title or location.
- Sort by price (Low → High / High → Low).
- "Not Available" badge for unavailable listings.
- Bookmark / favourite with localStorage persistence.
- Header with logo above brand name and total active listing count.
- Responsive grid layout.
